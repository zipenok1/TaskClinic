Результат нужно прикрепить архив файлом или ссылкой на гит или развернуть на своем сервере если есть у вас.
Стэк основной - php, sql(mysql)

1. Создать страницу с формой.
 В форме должны быть следующие поля:   
    - имя
    - email
    - телефон
2.   Создать базу данных с полями id name email phone
3. Реализовать отправку этой формы. Страница при отправке не должна перезагружаться (fetch/ajax).
4. Реализовать обработку запроса на php.    
Реализовать защиту от sql иньекций 
В обработчике нужно:
1) провести валидацию
    - email
    - телефон
2) Провести проверку есть ли в этой таблице элемент с заполненным именем, емейлом или телефоном, если есть и заявка уходила в период 5 минут, форму не отправлять!
3) При успешной проверке - форма должна скрываться, а пользователю должно выводиться сообщение об успешной отправки заявки.
4) При неудачной проверке - пользователю должна выводиться ошибка над формой.
